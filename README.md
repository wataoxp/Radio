# Radio
 
## Overview
FMReceiver IC RDA5807FP and LCDcontroller SPLC792A for STM32

## Requirement
* STM32CubeIDE
* STM32G030F6P6(Cortex-M0+)
* Clang

## Description
lowlayer(LL)I2C use.
When using another MCU
* Test with the HAL library first
* Input control differs for each MCU, so please change it.
